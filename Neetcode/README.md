# DSA Practice

Topic-wise log of DSA problems solved, with pattern + complexity for quick
revision. Each topic folder has its own `README.md` (the actual index) and a
`solutions/` folder with code.

## How this repo works

1. Pick a topic → pick the next unsolved problem from that topic's README.
2. Solve it, save code as `solutions/NN-problem-slug.py`.
3. Fill in the row in that topic's `README.md` (pattern, time, space, status).
4. Commit with a message like: `Array: solved Two Sum (Hashmap, O(n)/O(n))`
5. Push.

## Conventions

- **File naming**: `NN-problem-slug.py` — number matches the order in the
  topic README, slug matches the LeetCode URL slug.
- **Status column**: ⬜ Not started · 🟨 In progress · ✅ Solved & reviewed
- **Language**: pick one (Python used in templates here) and stick to it
  unless a problem specifically calls for another.
